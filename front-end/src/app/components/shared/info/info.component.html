Information about this application

This application was developed in the Angular framework for the User Interface. It uses the Material framework for easy CSS.

This UI communicates with a .NET API, which stores the data in a SQL Server database, hosted on a personal machine. Therefore, it is requested that you do not spam this application to avoid overstressing said personal machine. Doing so can result in an IP ban.

For the code of both the front- and back-end, please take a look on Github!
